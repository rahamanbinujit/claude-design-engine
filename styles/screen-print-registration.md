## Concept
The design is framed as a screen print registration sheet — the technical proof document used in serigraphy to verify color alignment, ink density, and print quality before a full run. The "artwork" (Rahaman's content) sits in the center as the print being proofed, surrounded by the technical apparatus of the printing process.

## Typography
- **Headline:** Space Grotesk 700, 72px, -2px letter-spacing, 0.95 line-height (the "artwork" type)
- **Stats:** Space Mono 700, 36px (monospace precision for numbers)
- **Technical specs:** IBM Plex Mono 300-400, 8-11px, 1.5-2px letter-spacing, uppercase (production metadata)
- **Name/title:** Space Grotesk 600, 16px / IBM Plex Mono 400, 10px

## Key Elements
1. **Registration crosshairs** (4 corners) — circle + crosshair lines
2. **Trim marks** — L-shaped corner cuts at print area edges
3. **Ruler markings** — numbered ticks along top and left edges
4. **Density bar** — gradient opacity steps (black + accent color) on right margin
5. **Color separation swatches** — squares with ink name, hex, mesh count, method
6. **Technical specs row** — substrate, method, registration tolerance, ink system, run size
7. **Chromist note** — italic handwritten-style approval annotation
8. **Halftone overlay** — subtle dot pattern inside print area (3% opacity)
9. **Misregistration effect** — 2px offset color layer at 8% opacity

## Rules
- Print area is ALWAYS dark (#1A1A1A) — the artwork
- Surrounding area is ALWAYS light cream — the registration sheet
- Only ONE color accent inside the print (red on a single element)
- Technical specs must be AUTHENTIC to screen printing (mesh counts, registration tolerances, ink systems)
- Edition should be "01 / 01" — emphasizing one-of-a-kind
- Chromist note adds human touch to technical document
