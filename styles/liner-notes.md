# Vinyl Liner Notes Style

> Part of the [Claude Design Engine](https://github.com/rahamanujit/claude-design-engine) style library. Adapt colors and fonts to your `brand.config.json`.


## Hierarchy (4 tiers from research)
1. Album title (Libre Franklin 32px/900, uppercase)
2. Track listing + credits (Libre Baskerville 12-13px, two columns)
3. Personal note (Libre Baskerville 12px italic)
4. Label info/legal (JetBrains Mono 8px, 12-15% opacity)

## Key Elements
- B&W photo top 30% with bottom-only fade
- Two-column body (tracks left, credits right, thin divider)
- Track format: `01  Track Name  4:32`
- Credits grouped by role with capability counts
- Personal note as closing thought
- Catalog number + label info at bottom

## Best For
- "Behind the scenes" / process posts
- Credits/acknowledgment content
- Portfolio/capability breakdowns
