# Recipe Card / Cookbook Style

> Part of the [Claude Design Engine](https://github.com/rahamanujit/claude-design-engine) style library. Adapt colors and fonts to your `brand.config.json`.


## Palette
- **Paper:** Warm vanilla `#FAF6F1`
- **Text:** Espresso brown `#3A2E28`
- **Accent:** Saffron `#D4883A`
- **Secondary:** Olive green `#6B7D4A`
- **Tertiary:** Paprika red `#C04830`
- **Notes:** Pencil gray `#8A8278`

## Content Mapping (the metaphor)
| Recipe Element | Maps To |
|---------------|---------|
| Category | "From the Kitchen" |
| Title | The system/concept |
| Prep Time | How long it took |
| Yield | Number of skills |
| Serves | Number of brands |
| Difficulty | Output multiplier |
| Ingredients | Individual skills with counts |
| Pencil Note | The secret/insight |
| Method | 3-step process |
| Total Yield | Final stat |

## Typography
- **Title:** Playfair Display, 56px, weight 700. Saffron italic on accent words.
- **Subtitle:** Playfair Display, 18px, italic, 40% brown
- **Info values:** Playfair Display, 22px, weight 600
- **Ingredient amounts:** Playfair Display, 16px, weight 600, saffron
- **Ingredient names:** Nunito Sans, 14px, weight 400, 65% brown
- **Method steps:** Nunito Sans, 14px, weight 300, 55% brown. Bold names. Olive/paprika accents.
- **Step numbers:** Playfair Display, 28px, weight 700, 10% brown (ghost numbers)
- **Pencil notes:** Caveat, 20px, pencil gray, rotate(-2deg)
- **Section headers:** Nunito Sans, 11px, weight 700, 4px letter-spacing, 30% brown

## Ingredient List Format
```
[saffron amount] [description with personality]
18              AI systems, freshly integrated
14              Content & copywriting skills, aged
2               Sales pipeline systems, slow-roasted
```
Add personality to descriptions: "freshly integrated", "aged", "finely tuned", "to taste", "slow-roasted"

## Handwritten Note
- Font: Caveat, 20px
- Color: pencil gray #8A8278
- Rotation: -2deg
- Include wavy underline on key word
- SVG pencil arrow pointing from note to relevant section

## Best For
- "Here's the recipe" breakdown posts
- Process/method content
- Skills inventory with personality
- Any post where the content naturally has ingredients → process → output
- The most shareable format -- people love recipe-style breakdowns
