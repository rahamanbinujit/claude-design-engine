# Postage Stamp Collection Style

> Part of the [Claude Design Engine](https://github.com/rahamanujit/claude-design-engine) style library. Adapt colors and fonts to your `brand.config.json`.


## Key Learning
Perforations need to be MORE visible — they're the signature element. Use stronger contrast or larger dots next time.

## Stamp Structure
- Country name (Cinzel 10px, caps, letter-spacing, top)
- Icon/illustration (center, 48px emoji or SVG)
- Denomination (Cinzel 42px, bold, color-coded)
- Unit label (Inter 10px, caps)
- Description (Inter 11px, 35% opacity)

## Best For
- Collection/inventory posts
- "Complete series" messaging
- Skill/achievement showcase with visual categorization
