# Data Journalism Style

> Part of the [Claude Design Engine](https://github.com/rahamanujit/claude-design-engine) style library. Adapt colors and fonts to your `brand.config.json`.


## Palette
- **Background:** Warm off-white `#FAFAF8`
- **Text:** Near-black `#1A1A1A`
- **Accent:** Signal red `#D63B30`
- **Data primary:** Blue `#2563EB`
- **Data secondary:** Gray `#94A3B8`
- **Rules/Dividers:** Light gray `#E2E2DE`
- **Bar track bg:** `#F0EFEB`

## Typography
- **Headline:** Source Serif 4, 48px, weight 700, -0.02em. Reads like a newspaper headline.
- **Deck:** Source Serif 4, 20px, weight 300, #555555
- **Byline:** Roboto, 12px, weight 500
- **Chart title:** Roboto, 14px, weight 700
- **Bar labels:** Roboto, 12px, #555555
- **Bar values:** Roboto Mono, 11px, weight 700 (white inside, black outside)
- **Big number:** Source Serif 4, 96px, weight 700, red
- **Stats:** Source Serif 4, 36px, weight 700
- **Body:** Source Serif 4, 16px, #333333, line-height 1.7
- **Methodology:** Roboto, 9px, #94A3B8

## Layout
- 1080x1350, off-white bg
- Red 4px top accent line (full width)
- Section tag (red) + date (gray) row with rule below
- Serif headline spanning 75% width
- Byline with rule below
- Horizontal bar chart (7 rows, descending order)
- Big number callout with 2px black top border
- 4-column stats row with vertical dividers
- Body text (serif)
- Methodology note with rule above
- Footer with name + style label

## Chart Rules
- Bars: 28px height, 2px border-radius
- Red for #1 category, blue for mid-range, gray for lower
- Values INSIDE bars (white text) when bar > 30% width
- Values OUTSIDE bars (black text) when bar < 30% width
- Descending order always
- Track bg: subtle off-white (#F0EFEB)

## Best For
- "Here's the data" proof posts
- Skills/achievement breakdowns with numbers
- Comparison posts (one person vs team)
- Any content that benefits from journalistic credibility
- Data-driven positioning where the FORMAT sells the message
